## 技术架构

### 多平台适配

这个应用程序使用golang和dart(flutter)作为主要语言, 可以兼容Windows, linux, MacOS, Android, IOS

使用了不同的框架桥接到桌面和移动平台上

- go-flutter => Windows / MacOS / Linux
- gomobile => Android / IOS


### 构建环境

- [golang](https://golang.org/) (1.16)
- [flutter](https://flutter.dev/) (2.10.3)
